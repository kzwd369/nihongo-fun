## 4. Differentiation Axes (Both Markets)

### Axis 1: Wasei-Eigo Category Ownership (PRIMARY)
- **zh-Hant**: 2-3 listicle competitors, no dedicated reference
- **es**: ~4 shallow articles, zero dedicated reference
- **Our play**: Build THE definitive wasei-eigo reference in both languages. Individual word pages with DefinedTerm schema. First-mover advantage = category ownership.

### Axis 2: Onomatopoeia Depth Beyond Listicles
- **zh-Hant**: 時雨の町 has dictionary entries; no one has editorial depth
- **es**: Japonés en la Nube has one "guía definitiva"; can be surpassed
- **Our play**: Categorized by emotion, weather, action, texture, food, manga SFX. Visual examples. Individual pages per word.

### Axis 3: E-E-A-T Through Real Experience
- **Both markets**: ZERO competitors use first-person travel/experience content
- **Our play**: Every article includes a real anecdote from 43-country travel experience. "I heard this at a konbini in Osaka." "My Taiwanese friend laughed when I used this wasei-eigo." Impossible for AI content farms to replicate.

### Axis 4: Spanish/Portuguese Loanwords in Japanese (Identity Bridge)
- **es**: Almost no coverage of パン (pan), カステラ (castella), タバコ (tabaco), etc.
- **zh-Hant**: Can cover unique Taiwan-Japan linguistic bridges
- **Our play**: "Your language is already in Japanese!" -- powerful identity hook for engagement

### Axis 5: Modern Technical SEO
- No competitor in either market uses DefinedTerm structured data
- No competitor has proper hreflang between zh-Hant/es/en
- Astro + Cloudflare Pages = top-tier Core Web Vitals
- Clean internal linking pyramid with pillar pages
